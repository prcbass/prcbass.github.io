Delivery Fee Portion Calculator

After recently attending GitHub Universe, I learned about an interesting desktop application development framework called Electron. This framework greatly interested me since I have created other personal projects in Javascript-based web development frameworks (MEAN JS) and as such could easily use some of those previous experiences as a basis for my work. As a busy college students, I sometimes find myself ordering delivery alongside my roommates to provide a meal. The problem comes in when the fees that are associated with delivery services then need to be split amongst the involved parties. This personal project serves as an introduction to development with Electron and provides a way for users to enter basic information with regards to a delivery order (price per item ordered, total delivery fees, tip, and tax). The result is the price each person should pay to evenly distribute the costs of ordering food. The application currently only has the main logic to calculate the shared fee portions and my next focus will be on an improved user-experience and the creation of a deployable application for use on Mac, Linux, and Windows.

  • GitHub Link: Delivery Fee Portion Calculator
  • Date: Present
  • Languages/Framework: Javascript (Node.JS, Electron)

Online Course Management Bot

I attended hackGSU Fall 2016 as my first hackathon and worked alongside 2 other UF students. Per my suggestion, my team decided to create a Slack integration for UF's online learning management system, Canvas. The result of our work is a Slack App with custom slash commands that allow students to obtain information regarding their assignments, registered courses, course calendar, online profile, and more from their Canvas account. Our Slack App (under the name "Online Course Registration Bot") is hosted on a Node.JS web server and calls out to Instructure's REST API to obtain relevant user information. While the application currently requires local hosting, it allows administrators on any Slack team to add our integration. Current planned features include deploying our application to AWS, working with Instructure and UF to obtain a non-user specific developer access token, and more custom commands.

  • Date: Present
  • Languages/Framework: Node.JS
